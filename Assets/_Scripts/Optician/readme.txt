In this folder you can find differents scripts usefull for the OpticianCalibration Scene : 
----------------------

FOVPoints is a Serializable class to describe objects for the scriptable objects

fovScriptableObject is a ScriptableObject to store data relative to eye-lens distance, like eye-lens distance and the corresponding FOV points.

OpticianController is the script that holds all the logical for the OpticianCalibration Scene.

PointingSystem is the script that holds all the logical for the pointing system of the controller and is used to add points to different lists.
